How to install Dark Reader for mobile devices

QR Code

Simply scan the following QR code with your phone.

You will be prompted to install our iOS app from the App Store, or Microsoft Edge browser for Android that will include our extension.

iOS and iPadOS

Download Dark Reader from the App Store. iOS 15 is required.

Dark Reader for iOS has an amazing brand new design. You can choose from many color presets, or set lovely text and background colors of your choice.

Android

Microsoft Edge

Microsoft Edge for Android supports Chromium extensions.

  • Scan the following QR code with your Android phone:
  • You will be redirected to Google Play. Click Install.
  • Follow the initial setup. Dark Reader should be included by default.

If you already have Edge for Android installed, go to Extensions menu and install Dark Reader from the list.

If you haven't been redirected automatically, simply install Edge from the Play Store.

Firefox

Mozilla Firefox for Android supports mobile extensions.

  • First you need to install Firefox for Android.
  • Open the browser, click Menu icon (three dots in the bottom-right corner).
  • Click Add-ons, find Dark Reader in the list, click + (plus icon).
  • Alternatively you can visit Dark Reader's page in the Firefox Add-ons store.
  • Now click Menu icon, Add-ons. Dark Reader will appear there.
